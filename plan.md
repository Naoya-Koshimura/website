『直手有為のページ』の制作計画

概要
私（直手有為）のインディーゲーム作品をまとめたWebサイトを作る

要件
コンテンツ：自己紹介、作品のリストと紹介記事、ブログ、コンタクト、Xやsteamへのリンク
更新頻度：多い月で数十回程度
セキュリティ対策（暫定）：Cloudflare Pagesで静的サイトを公開する
重視する要素：なるべく更新に人的リソースがかからないこと（ただし、コーディングはあなたに任せられるので、コーディング時間は含めないものとする）

実装プラン
1. 基本環境構築 [完了]
    - Astro + Tailwind CSS
    - 共通レイアウト設計

2. ページ実装
    - Home: 自己紹介と最新情報 [完了]
    - Games: 作品一覧と詳細 [作業中]
        - Astro Content Collections を利用
        - Markdownで管理しやすくする
    - Blog: 開発日誌 [既存テンプレート活用]
    - Contact: 連絡先フォーム/リンク [未]

3. デプロイ準備
    - Cloudflare Pages用のアダプター設定（必要に応じて）
    - ビルド確認
